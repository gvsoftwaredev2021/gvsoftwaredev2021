There Should Be One H1 Tag in your assignment

SubHeadings can be the h2 & h3 tags

Date Submitted: 2021-10-04

Enclose a section between section tags


	<section>
		// section content goes here
	</section>

Enclose section titles with h4 tags


	<section>
		<h4>Section Title</h4>

		<p>Section text content.</p>
	</section>

Example:


	<section>
		<h4>This is section 1</h4>

		<p>Here is a a paragrah for section 1.</p>
	</section>

--------- How it will look ---------

This is section 1

Here is a a paragrah for section 1.

--------- How it will look ---------


Let's use only unorder lists 


	<p>List name</p>
	<ul>
		<li>List item 1</li>
		<li>List item 2</li>
	</ul>

--------- How it will look ---------

List name

  • item 1
  • item 2
  • item 3

If you need to show your code

Enclose your code in pre-format and code tags. Replace less than (<) and greater than (>) signs by "&lt;" and "&gt;"


	<pre><code>
		&lt;h4&gt;Example section title&/lt;h4&gt;
	</pre></code>

--------- How it will look ---------


	<h4>Example section title</h4>
